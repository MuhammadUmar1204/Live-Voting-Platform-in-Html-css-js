# 🗳️ Live Voting Platform (Frontend)

This project is a simple voting platform developed using HTML, CSS, and JavaScript. It allows users to vote for different options and instantly see the results.

## 🚀 Features
- User-friendly voting interface
- Dynamic vote counting using JavaScript
- Instant result updates without page reload
- Responsive design for mobile and desktop

## ⚠️ Limitations
- This is a frontend-only project
- Votes are not stored permanently
- No real-time multi-user synchronization

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📌 Future Improvements
- Add backend integration (Firebase or Node.js)
- Store votes in a database
- Enable real-time voting across multiple users
- Add user authentication system
